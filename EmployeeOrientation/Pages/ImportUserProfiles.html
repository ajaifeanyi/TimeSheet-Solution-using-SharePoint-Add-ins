Import User Profiles

Use the following steps to import user profiles from MS Excel spreadsheet
1. Select a range of cells in MS Excel: 6 columns as below and max. 50 records (per single import operation).
2. Copy selected cells by using the keyboard shortcut CTRL+C
3. Click on the first cell on below spreadsheet and use CTRL+V to paste records
4. If there is no validation error (cells marked by red background), click "Import" button.

Data validation rules
• The email must be registered in Sharepoint and user needs to login at least once to the Sharepoint Site
• All fields except "Second Approver Email" are mandatory

Important Notices
• Records with at least one cell marked by red color (validation error) will not be imported
